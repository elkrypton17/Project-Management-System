# Project-Management-System
A web-based project management system designed to streamline the communication, tracking, and evaluation process between academic supervisors and students for final-year projects.
Academic Project Management System

A centralized web-based platform designed to bridge the gap between academic supervisors and students, ensuring streamlined project tracking, communication, and evaluation.

Features

Supervisor Portal: Comprehensive dashboard to monitor student progress, project milestones, and pending requests.

Student Workflow: Intuitive interface for submitting project updates, uploading documents, and receiving feedback.

Centralized Database: Robust architecture to map students to specific supervisors and manage project lifecycle data.

Role-Based Access: Secure authentication ensuring that only authorized users access relevant project information.

Responsive Design: Fully mobile-responsive interface for access on any device.

Technology Stack

Frontend: HTML5, CSS3, Bootstrap (for a fast, responsive UI).

Backend: PHP (using PDO for secure database interactions).

Database: MySQL.

Prerequisites

To run this project locally, you will need:

A local server environment (like XAMPP, WAMP, or MAMP).

PHP 7.4 or higher.

MySQL database.

Installation

Clone the repository:

git clone [your-repository-url]


Setup Database:

Import the provided database.sql file into your MySQL database using phpMyAdmin.

Configure Connection:

Update the database configuration settings in your config.php file with your local database credentials.

Run:

Place the project folder in your server's htdocs or www directory and navigate to http://localhost/your-project-folder.
